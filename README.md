# IronCore Fitness - Gym Website & Calorie Tracker

Welcome to the repository for **IronCore Fitness**, a modern, responsive gym website built to help users achieve their fitness goals. Beyond standard gym facility showcases, this platform features a built-in **Calorie Intake Calculator** to provide personalized nutritional guidance.

## 🚀 Features

- **Personalized Calorie Calculator:** Calculates daily caloric needs (BMR and TDEE) based on:
  - Age
  - Height & Weight
  - Activity / Fitness Level
  - Specific goals (Bulking, Cutting, Maintenance)
- **Training Modules:** Dedicated sections for various training styles, including weightlifting programs and track event guides (like 400m sprint routines).
- **Modern UI/UX:** Fully responsive design ensuring a seamless experience across desktop and mobile devices.
- **Class Schedules & Membership:** View upcoming gym classes and explore membership tiers.

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/gym-website.git
   cd gym-website
   ```
2. **Install dependencies:**
   ```bash
   # For Node.js
   npm install  
   
   # Or for Python
   pip install -r requirements.txt
   ```
3. **Run the application:**
   ```bash
   # For Node.js
   npm start   
   
   # Or for Python
   python app.py
   ```

## 💻 Usage: Calorie Calculator

1. Navigate to the **Nutrition/Calorie Calculator** section on the homepage.
2. Enter your biometrics (Age, Height in cm, Weight in kg).
3. Select your current fitness level from the dropdown (e.g., Sedentary, Lightly Active, Moderately Active, Very Active).
4. Hit **Calculate** to receive your daily caloric target and a basic macronutrient breakdown (protein, carbs, fats).
